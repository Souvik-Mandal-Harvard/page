Start Here

Every project in LS100 follows the same arc — frame a question, learn the tools, collect data, analyze it, and communicate the result. This page maps the whole route; you’ll only ever focus on your next step.

Jump to your starting point¶

  • I’m new to coding → start with Module 00B · Python Fundamentals, Notebook 01.

  • I already know Python → skim 00B, then pick your data modality below.

  • I work with video → Module 01A · Video Data.

  • I work with audio → Module 01B · Audio Data.

  • I work with digital/tabular data → Module 02 · Data Science & Statistics.

The milestone path¶

  1. Frame your study — Research Guide 01: turn an interest into a researchable question.

  2. Build Python fluency — Module 00B, Notebooks 01–05.

  3. Choose your data modality — Video (01A) or Audio (01B), and run the processing pipeline. (This is the one fork in the path — both branches rejoin at analysis.)

  4. Analyze your outcomes — Module 02, using the statistics guide and the decision-tree poster to pick the right test.

  5. Communicate your findings — Research Guide 02 · Art of Communicating Science.

Tip

Stuck on a term? See the Essential Python Terminologies guide. Unsure which test to use? See the statistics decision tree.
